Rede Social para Músicos

Desenvolvido em 2024 como nosso projeto de TDS, o GrooveClub é uma plataforma inovadora que conecta músicos e fãs em um ambiente colaborativo.

A rede social permite que artistas compartilhem suas músicas, encontrem colaboradores, participem de comunidades específicas por gênero musical e interajam com seus fãs de forma direta.

Principais funcionalidades implementadas:

  • Perfis profissionais
  • Sistema de postagem
  • Player de áudio integrado
  • Sistema de curtidas
  • Cadastro personalizado

Tecnologias Utilizadas

  • Backend: Python com Flask
  • Frontend: React
  • Banco de Dados: MySQL

Groover - Editor Musical Integrado

Nosso projeto atual para PDS 2025

Este ano estamos desenvolvendo o Groover, um editor de música completo que será integrado diretamente ao GrooveClub, transformando a plataforma em um verdadeiro estúdio virtual.

O Groover permitirá que os usuários criem, editem e compartilhem músicas diretamente na plataforma, sem necessidade de softwares externos.

Funcionalidades planejadas para o Groover:

  • Editor de partituras e arranjos musicais
  • Gravação e edição de áudio
  • Biblioteca de samples e loops
  • Colaboração entre usuários
  • Exportação em formatos profissionais
  • Integração direta com o perfil do GrooveClub

Tecnologias que Utilizaremos

  • Web Audio API + Tone.js
  • React
  • Otimizações no banco de dados
